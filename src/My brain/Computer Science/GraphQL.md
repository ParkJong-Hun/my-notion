# GraphQL

<aside>
💡

Facebook이 2015년 공개한 데이터 쿼리 언어.
클아이언트가 서버로부터 필요한 데이터만 효율적으로 요청하고 받을 수 있도록 설계.

</aside>

### 특징

- **클라이언트 주도**
- 싱글 엔드포인트
- 타입 시스템
- 계층 구조
- 서버 버전 관리 불필요
- 실시간 구독 기능

### 기본 요소

**Query**

서버에 데이터를 요청하는 방식.

**Mutation**

데이터 생성, 수정, 삭제 요청.

**Subscription**

실시간 데이터 요청.

**Schema**

서버가 제공하는 데이터 구조, 타입 정의.
