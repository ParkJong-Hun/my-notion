# Mustache

<aside>
💡 논리 없는 템플릿 언어.
HTML/XML 같은 데이터를 표현하는데 주로 사용됨.
특정 언어에 종속되지 않으며 다양한 프로그래밍 언어로 구현됨.

</aside>

### 특징

- 언어 불가지론적
- 단순함
- 유연성

# 기본 문법

`{{}}`

- 변수의 값 반환.

`{{#}}내용{{/}}` 

- 섹션.
- 루프나 조건을 적용할 때 사용.

`{{^}}내용{{/}}` 

- 인버티드 섹션.
- 데이터가 없을 때 실행.

`{{> }}` 

- 파샬.
- 재사용할 수 있음.

`{{! }}` 

- 코멘트.
- 주석 작성.
