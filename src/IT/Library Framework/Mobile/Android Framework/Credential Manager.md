# Credential Manager

<aside>
💡

유저 이름, 패스워드, 패스키, 제휴 로그인 솔루션(ex/ 구글 계정으로 로그인) 같은 멀티 로그인 방식을 지원해 개발자를 위한 통합을 간소화하는 Jetpack API.

</aside>

<aside>
⚠️

패스키는 Android 9 이상에서만 지원.

</aside>

**isCredential**

- 사용자 인증 정보 필드를 표시하는 UI의 속성.
- Credential Manager나 서드 파티 사용자 인증 정보 제공업체와 호환되는 것임을 의미.
- 자동 완성 서비스를 더 좋게 지원.
