### Android 버전 별 특징

Android 6.0 (API 23)

- 런타임 권한 등장.
- 앱 설치 시 모든 권한을 허용하지 않고 일부 권한을 유저에게 요청,.
- 허용, 거부가 표시.

Android 10.0 (API 29)

- “앱을 사용하는 동안만 허용” 추가.

Android 11 (API 30)

- “이번만 허용” 추가.

Android 13 (API 33)

- 특정 사진, 비디오만 선택 공유 추가.
