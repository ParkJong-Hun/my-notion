# AppDelegate

<aside>
💡

UIApplicationDelegate 프로토콜을 따르는 객체로, 앱의 주요 라이프사이클 이벤트를 처리.

</aside>

**application()**

- 앱 시작 시 호출

**applicationWillResignActive()**

- 앱 비활성 상태 전환 시 호출

**applicationDidEnterBackground()**

- 앱 백그라운드 전환 시 호출

**applicationWillEnterForeground()**

- 앱 포그라운드 전환 시 호출

**applicationDidBecomeActive()**

- 앱 활성화 상태 시 호출

**applicationWillTerminate**

- 앱 종료되기 직전 호출
