**Local State**

- 특정 컴포넌트 내에서만 데이터 관리.
- 함수형 컴포넌트에서는 리액트 hook과 props를 사용하는 것을 의미.
- 앱 크기가 커질수록 하위 컴포넌트에 데이터를 전달하기 위해 props drilling이 증가해 유지보수가 어려워짐.

**Global State**

- 프로젝트 전체에서 사용가능하도록 상태를 총괄 관리.
- Redux, Context API 등이 존재.
- 유지보수가 간단하지만, 상태관리 오류시 사이드 이펙트가 커짐.
